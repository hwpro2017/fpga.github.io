在Windows下使用ISE

发表于 2017-02-06 | 分类于 工具 |

赛灵思ISE开发环境是很多老玩家一直以来在使用的FPGA开发环境。这几年PC OS逐渐升级到了Windows10时代，众多玩家们的机器也升级到了win10；然而ISE在Win10下却不能正常战斗了，这也一度成为玩家们拒绝升级OS的理由。Win10下真不能用ISE了么？WIN10下真不能用Planahead了么？
当然不是，下面我们就来列举一些方案，如何在Windows10下使用ISE以及Planahead。

官方补丁

几年前官方宣布ISE将会停止更新，后续的新器件与特性将只在Vivado工具中进行支持。虽然ISE工具的版本号永远的停在了14.7，但是有小道消息表明赛灵思会做升级补丁以便让各位玩家能够在Win10下流程运行ISE工具。
小道消息如图：

不过截止目前，还没有进一步的官方消息释放。
各位玩家继续期待吧。

手动修改

在Win10 Win8.1下使用ISE 14.7

ISE 14.7版本的工具在Win10下面安装完成后，每当“打开工程”，“加载License”等操作时，ISE会报错崩溃而无法正常使用。
我们可以根据下面的步骤来进行修改来解决问题：

定位至路径：

\Xilinx\14.7\ISE_DS\ISE\lib\nt64\
文件重命名：
将”libPortability.dll” 改为 “libPortability.dll.orig”
文件重命名：
将”libPortabilityNOSH.dll” 改为 “libPortability.dll”

定位至路径：

\Xilinx\14.7\ISE_DS\common\lib\nt64\
文件重命名：
将”libPortability.dll” 改为 “libPortability.dll.orig”
文件重命名：
将”libPortabilityNOSH.dll” 改为 “libPortability.dll”

接下来就可以在Win10 Win8.1下正常使用ISE 14.7 啦！

在Win10下使用 PlanAhead 14.7

虚拟机环境

最直接的方案，当然是安装在对应的匹配操作系统中咯。
在Win10下，安装VirtualBox或VMware等虚拟机环境，安装Win7虚拟机来解决问题。
