Vivado license 申请指南

发表于 2017-02-08 | 分类于 工具 | | 次

想用Vivado工具但是没有license怎么办？
想用SDSoC工具但是没有license怎么办？
License应该如何申请？如果加载？
有需求的玩家们请关注正文。

Xilinx License申请指南

首先，申请License需要有Xilinx官网账号；

没有账号的玩家们可以在这里申请：Xilinx官网；

接下来登录Xilinx许可解决中心，如图所示；

点击箭头处的“Xilinx 产品许可网站” （该步骤需要登录Xilinx官网账号），进入如下界面；

在Certificate Based Licenses窗口中勾选所需License，然后点击下方按钮：

这时会弹出如下窗口：

在窗口中填写对应信息

（1）为所选License；（3）可选填注释；
（2）为玩家们需求填写的系统信息；点击“Selece a host”-“Add a host” 填入机器信息；其中host name处填写任意主机名称，下方选择对应操作系统以及host id type（通常选Eth MAC），在最后填写本机对应的MAC；

其中，ETH MAC可通过 CMD中运行以下代码来查看主机网卡MAC地址：

另外，可通过Xilinx 许可证管理器（VLM）来进行查看，VLM可从程序菜单进入：


在VLM中，可通过点击View Host Information，来查看主机信息，以便在申请license时填写：

继续生成License

点击next，进入（4）REVIEW LICENSE REQUEST界面：

确认无误继续点击next，随后会弹出如下成功提示，此时所需license会自动发送至玩家在官网注册账号的邮箱中：

申请IP License

此外，玩家们还可点击“search now”来打开搜索窗口，在这里可以按照需求添加Xilinx提供的可评估IP来产生对应的License：

在上述窗口中勾选对应IP，并且点击ADD添加，随后步骤和上文中一致。

License加载

在获得License文件之后，我们可以打开VLM，选择Load License- Copy License来将我们获得的License文件导入系统。

愉快的玩耍吧！
