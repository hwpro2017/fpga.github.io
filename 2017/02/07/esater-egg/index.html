Vivado安装包-最小体积版本

发表于 2017-02-07 | 分类于 工具 |

这里给出的是博主专为 FPGAPlayer.com 的初级玩家们精心压制的超小容量Vivado安装包。

完整版Vivado安装包

我们来看一下，如果选择全部完整安装的话。需要吃掉多少硬盘空间吧

是的噢，65G！！！
即便是安装完后把安装包删掉，那也还有接近50G！！

这里附上完整版下载与安装流程：Vivado工具入门-下载与安装

因此，为便于初学者在使用之初不必下载数十G的安装文件，以及不必安装完全不需要的组件。FPGAPlayer.com为各位玩家们给出了最小体积版本的Vivado安装方案。

那么玩家们应该挑选哪个版本下载呢？

如果你需要使用到以下器件中的某一个

  • Artix®-7 (XC7A15T、 XC7A35T、 XC7A50T、 XC7A75T、 XC7A100T、 XC7A200T)
  • Kintex®-7 (XC7K70T、 XC7K160T)
  • Zynq®-7000 All Programmable SoC(XC7Z007S – XC7Z7030)

请选择Vivado_2016.4_WebPack_ZYNQ_A7_K7_HLS_noSDK

  • 安装包容量： 2.45G
  • 安装后容量： 7.5G以内

如果你需要使用到以下器件中的某一个

  • Zynq®-7000 All Programmable SoC(XC7Z007S – XC7Z7030)

请选择Vivado_2016.4_WebPack_SDK_only

  • 安装包容量： 2.68G
  • 安装后容量： 8.5G以内

如果你还需要使用SDK

  • Artix®-7 (XC7A15T、 XC7A35T、 XC7A50T、 XC7A75T、 XC7A100T、 XC7A200T)
  • Kintex®-7 (XC7K70T、 XC7K160T)
  • Zynq®-7000 All Programmable SoC(XC7Z007S – XC7Z7030)

请“！！额外！！”选择Vivado_2016.4_WebPack_SDK_only

  • 安装包容量： 1.55G
  • 安装后容量： 9.5G以内

是的，你没看错，如果要用SDK的话，需要额外再使用近10G的硬盘空间来安装SDK工具；
SDK工具安装过程中解压出接近7G的GUN 工具链，所以没办法啦，已经是最小了。

如果你需要使用到以下器件中的某一个：

  • Zynq UltraScale+™ MPSoC (XCZU2CG/EG, XCZU3CG/EG)

请等待Vivado 2017.1版

如果上述器件不涵盖你所需要使用的器件

  • 那么你一定不算是初级玩家了，请下载System版本吧，如需帮助请留言联系。

安装流程

  1. 解压安装包
  2. 打开xsetup.exe
  3. 点击next
  4. 勾选三个I Agree
  5. 连续点击next到选择安装路径界面
  6. 选择安装路径
  7. 再次点击next进行安装
  8. 等待安装完成

最后附上获取方式以及声明

获取方式

  1. 扫描下方二维码
  2. 点击关注
  3. 菜单点击“便民服务”-“Vivado安装包”
  4. 获取下载链接
  5. 安装/使用请关注fpgaplayer.com

上述Vivado安装文件来自XILINX官网，FPGAPlayer.com仅针对安装组件重新配置选择后压缩，以便于初学者在使用之初不必下载数十G的安装文件，以及不必安装完全不需要的组件。以上安装文件版权不属于FPGAPlayer.com，如有相关人士认为FPGAPlayer.com做法不妥，请联系hwpro2017@163.com。
